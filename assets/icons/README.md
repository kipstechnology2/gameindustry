# PWA & Touch Icons

Required asset list (replace placeholders before going live):

| File | Size | Purpose |
|------|------|---------|
| `icon-32.png` | 32×32 | Browser tab favicon |
| `icon-180.png` | 180×180 | Apple touch icon |
| `icon-192.png` | 192×192 | Android home screen |
| `icon-512.png` | 512×512 | Android splash / store |
| `icon-maskable-192.png` | 192×192 | Maskable (Android adaptive) |
| `icon-maskable-512.png` | 512×512 | Maskable (Android adaptive) |
| `og-banner.png` | 1200×630 | Open Graph / Twitter card |

Maskable icons should reserve a **safe zone** of about 80% diameter
(see https://web.dev/maskable-icon/).
